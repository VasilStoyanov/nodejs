# Project Agatha

### Steps to run the project:
1.  `npm install` (skip this if you have already done it)
2.  `npm start`

### Technologies to be used:
*   NodeJS (ExpressJS)
*   MongoDB
*   Reactive extensions for JS (RxJS)
*   Webpack
*   Babel
*   ECMAScript 2016
*   Axios
*   Material lite
*   ESLint (Configure in your favorite IDE/editor)
*   GraphQL

### Optionally use:

*   React JS
*   Docker

### Unit testing:

*   Mocha + Chai
